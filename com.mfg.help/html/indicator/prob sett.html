Indicator Settings

As we do with the Indicator Settings we can also change the probabilities Settings by opening that tab of the editor:

General Parameters

·         Patterns Calculation Scale will be the scale used to compute the patterns containing a similar number of belonging elements.

·         Bounds Step is the step we will use for the bounds of the intervals in the pattern Generation.

·         Targets Step is the step we will use for the targets defined in the probabilities calculation.

·         Min Matches % is the minimum percent of elements allowed for an interval to be considered for the calculation of the T1 for infinite bounds patterns.

·         SC Filter is the mode we can choose to compute the SC touch probabilities.

Multi Scales Parameters

·         Multiple Scales tells if we will compute the probabilities based on the higher scales clusters and direction, or not.

·         Scales Number is the number of scales considered in the multiple scale versions.

·         Cluster Size is the size of the split we do for the [0,1] probabilities range to determine the corresponding cluster for a probabilities value.

Pattern Generation Parameters

For each ratio pair we can select whether if it will be considered or not by flagging it or not and we can also specify in how many intervals this ratio values will be split.
