Probabilities Perspective 

After creating a symbol and with it a new indicator configuration we can open its editor when we can Create Probabilities:

When we execute such command the calculation starts and after the required time, when we finish computing the probabilities the Probabilities Perspective is automatically opened:

By selecting a row on the Probabilities Set view, we can see the corresponding Targets Distribution for that Key. And Also when we select one row of the Patterns view that correspond to a pattern that has Upper bound equal to infinite, in the T1 Computation view it is shown the details of the corresponding computation of the T1.

Views on the perspective

 

1.       Patterns View:

                                 i.            Shows the patterns with its bounds, the max sw0 ratio, the number of elements, the T1s,…

                               ii.            When we select a pattern with Infinite (Inf) Upper Bound (U) the statistics of the T1  calculation of that pattern are shown on T1 Computation View.

2.       T1 Computation View:

                                 i.            Shows the intervals Upper bounds, the matches for the interval, the matches %, number of elements that reached the upper bound, and the probability (reached/matches)

                               ii.            Dark gray rows are not considered for having a low matches %

3.       Probabilities Sets View:

                              iii.            Shows the sets that we considered on the calculation of probabilities, with its PID, the scale, the Base Scale (BSc), the Cluster ID (CID) and the dir (C/NC) relative to the HS

                             iv.            When one row is selected shows on Targets Distribution view the statistics for each target.

4.       Targets Distribution view:

                               v.            Shows for each TID the target value, the number of times it was reached, the number of times it was not reached, the total (reached + not reached), and the probability for that TID.

5.       SCT Probabilities Table

                             vi.            Shows the statistics for the sc touch probabilities calculations:

                            vii.            the scale, the order of the SC touch, the Base scale Cluster  (BSC), the TH count (number of times that ends up in a new TH), the number of total times the sc touch configuration appears, the TH probability and the HHLL probability (1 – TH Prob)

6.       Probabilities Log

                          viii.            With the events of the calculations.
